ARTH 105 Quiz 3 - Art History Learning Tool
=============================================

QUICK START GUIDE:

1. SETUP:
   - All code is contained in index.html (single file)
   - Place your artwork images in ./images/ subdirectory with these names:
     * img-000.png = Kanishka Reliquary
     * img-001.png = Hoti Mardan Buddha
     * img-002.png = Fasting Buddha
     * img-003.png = Bodhisattva Maitreya
     * img-004.png = Yakshi with mirror (Bhuteshar)
     * img-005.png = Yakshi putting on clothes (Bhuteshar)
     * img-006.png = Katra Buddha
     * img-007.png = Bala Buddha of Year 3
     * img-008.png = The goddess Hariti with three children
     * img-009.png = Bodhgaya Buddha of Year 64

2. OPENING THE TOOL:
   - Open index.html in any modern web browser (Chrome, Firefox, Safari, Edge)
   - No server or internet connection required
   - All functionality is self-contained

3. LEARN MODE:
   a) Terms Tab:
      - Search for any of the 29 art history terms
      - Click a term to view its full definition
      - See related terms and view associated images
      - Navigate between related terms with clickable chips
   
   b) Artworks Tab:
      - View all 10 artworks in a grid
      - Click any artwork to open a detailed modal
      - See full image, description, and key concepts
   
   c) Timeline Tab:
      - See chronological layout of art periods
      - Pre-Kushana, Kushana (split by Mathura/Gandhara), and Gupta eras
      - Click any item to view its term details
      - Read Key Insight about stylistic evolution

4. QUIZ MODE:
   a) Configuration:
      - Select question types: Flash Cards, Fill-in-the-Blank, Image Questions
      - Choose number of questions: 10, 20, 30, or All
      - Click "Start Quiz"
   
   b) During Quiz:
      - Flash Cards: Click to flip, mark "Got it" or "Still learning"
      - Fill-in-the-Blank: Type your answer and submit
      - Image Questions: Choose the correct option
      - Track progress with progress bar and score badge
   
   c) Results:
      - View final score and grade (A/B/C/D/F)
      - See breakdown of all questions with answers
      - Retake same quiz or create a new one

5. FEATURES:
   - Real-time search filtering
   - Responsive mobile design
   - CSS 3D flashcard animations
   - Interactive modal overlays
   - Flexible answer matching (case-insensitive for fill-in-the-blank)
   - Comprehensive feedback on quiz results
   - No external dependencies or internet required

CONTENT INCLUDED:
- 29 Art History Terms (all Kushana/Gupta Buddhist art period)
- 10 Artwork Entries (with descriptions and concepts)
- 78 Total Quiz Questions:
  * 29 Flashcard questions
  * 19 Fill-in-the-Blank questions
  * 30 Image questions (3 per artwork)

BROWSER COMPATIBILITY:
- Chrome/Chromium: Full support
- Firefox: Full support
- Safari: Full support
- Edge: Full support
- Mobile browsers: Full responsive support

TECHNICAL DETAILS:
- Single HTML file (74 KB)
- Embedded CSS3 styling
- Vanilla JavaScript (no frameworks required)
- All data stored in JavaScript arrays
- Images loaded from ./images/ directory

For questions or issues, ensure:
1. All image files are in the ./images/ folder with correct names
2. File is opened as HTML (not as plain text)
3. Browser has JavaScript enabled
4. Images are in PNG format

Enjoy your studies!
